Aula 04 JS

Desafio Prático

Se o usuário escolher "Fazer Depósito", o programa deve solicitar o valor do depósito. Certifique-se de validar se o valor é numérico e maior que zero.
Se a validação for bem sucedida, atualize o saldo e exiba uma mensagem indicando o sucesso da transação.
Se o usuário escolher "Sair", encerre o programa exibindo uma mensagem de despedida. Se o usuário escolher uma opção inválida, exiba uma mensagem
indicando isso e permita que o usuário faça uma nova escolha.
